# 🏠 Roof

Modern responsive website for a roofing company built with **React.js**.

## 🌐 Live Demo

🔗 https://arturmk099-dev.github.io/Roof/

---

## ✨ Features

- ⚛️ Built with **React.js**
- 🌍 Multi-language support (4 languages)
  - 🇦🇲 Armenian
  - 🇷🇺 Russian
  - 🇺🇸 English
  - 🇩🇪 German
- 📱 Fully responsive design for all devices
- 🎞️ Image slider / carousel
- ✨ Smooth scroll animations
- 🎨 Modern and clean UI
- ⚡ Fast performance
- 🧩 Component-based architecture

---

## 🛠️ Technologies

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- React Router
- React Slick (Carousel)
- i18next
- React i18next

---

## 📱 Responsive Design

The website is optimized for:

- 📱 Mobile phones
- 📲 Tablets
- 💻 Laptops
- 🖥️ Desktop computers

---

## 🌍 Languages

The website supports four languages:

- Armenian 🇦🇲
- Russian 🇷🇺
- English 🇺🇸
- German 🇩🇪

Users can switch between languages instantly.

---

## 🚀 Installation

```bash
git clone https://github.com/ArturMk099-dev/Roof.git

cd Roof

npm install

npm start
```

---

## 📦 Build

```bash
npm run build
```
